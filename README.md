# OneDrive Linux Sync

Selective OneDrive sync client for Linux featuring a PySide6 desktop UI, Microsoft Graph authentication, and background scheduling via systemd.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -e .[dev]
```
